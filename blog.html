What are the different ways to select an element in the DOM?

We can select an element in the DOM by several ways. For Example:

  • Using getElementById
  • Using getElementsByClassName
  • Using getElementsByTagName
  • Using querySelector
  • Using querySelectorAll

What is the difference between innerHTML, innerText, and textContent?

  • InnerHTML: Using innerHTML allows you to see exactly what’s in the HTML markup contained within a string, including elements like spacing, line breaks and formatting.
  • InnerText: This approximates the “rendered” text content of a node and is aware of styling and CSS. It’s most effective when you only need to see what’s in the element without the formatting.
  • TextContent: This retrieves and sets the content of the tag as plain text. It’s most effective when you want to see what’s in an element, plus styling.

What is event delegation in the DOM?

Event delegation in the DOM is a technique where a single event listener is attached to a parent element, rather than individual listeners on each child element. This leverages event bubbling, where events originating from child elements propagate up the DOM tree to their parent.

What is event bubbling in the DOM?

Event bubbling in the DOM is the process by which an event, when triggered on a specific element, propagates upwards through the DOM tree. Starting at the target element where the event originated, the event triggers any attached event listeners, then travels to its immediate parent element, triggering those listeners as well. This upward propagation continues through each ancestor element, effectively "bubbling" up the hierarchy until it reaches the root of the DOM.
